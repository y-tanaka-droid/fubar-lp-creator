# /lp-create スキル

LP制作のフルフローをガイドします。以下の手順を順番に実行してください。

## Step 1: クライアント情報のヒアリング

ユーザーに以下を確認してください（まだ提供されていない場合）：

```
LP制作くんです！以下を教えてください：

【必須】
1. クライアント名（会社名 or 個人名）
2. 業種・サービス内容
3. ターゲット（年齢・職業・悩み）
4. 強み・実績（数値があると最高）
5. LINE友だち追加URL

【あると良い】
6. 参考にしたいLPのURL
7. ブランドカラー（なければFUBARデフォルトを使用）
8. 納品希望日
```

## Step 2: ナレッジ読み込み

`docs/LP制作くん_総合ナレッジベース.md` を必ず読んでから制作を開始。

## Step 3: LP構成案を3パターン提案

以下の3パターンで骨子を提案：
- **パターンA**: 問いかけ型（課題提示→共感→解決）
- **パターンB**: 実績先出し型（数値→理由→CTA）
- **パターンC**: ストーリー型（before→after→仕組み）

## Step 4: 選ばれたパターンでHTMLを生成

`templates/base_lp.html` をベースに、クライアント情報を埋め込んでHTMLを生成。

生成時の必須要件：
- レスポンシブ（375/768/1280px）
- Noto Sans JP（Google Fonts）
- スクロール fade-in アニメーション
- 画面下部固定CTAバナー
- LINE CTAボタン（緑 #06C755）
- OGP設定

## Step 5: 品質チェック

以下を確認してOKならファイルを保存：

```
□ FV に3秒で伝わるコピーがあるか
□ CTAが FV・中段・最下部の3箇所にあるか
□ LINE URLが全CTAに正しく入っているか
□ SP（375px）で表示崩れがないか
□ 実績数値が具体的か
□ 「絶対に成功します」等のNG表現がないか
□ OGP設定が入っているか
```

## Step 6: 納品ファイルを保存

```
output/[クライアント名]/index.html
output/[クライアント名]/README.md
```

README.mdには公開URL・LINE URL・修正メモを記載。
